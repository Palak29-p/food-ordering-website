# YummyGo - Food Ordering Website

## Project Description
YummyGo is a food ordering website developed to provide users with an easy way to explore food items and view different food categories. The project includes a user-friendly frontend interface and a backend server for handling API requests.

## Features
- Attractive food ordering website design
- Food categories like Pizza, Burger, Pasta, Cakes, Drinks, etc.
- Menu page with food items
- Cart page
- Login page
- Backend API integration
- Responsive user interface

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
